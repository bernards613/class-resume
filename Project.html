Parcel Marshall:

UPS Driver Assistant Application using TypeScript that allows a user to add delivery addresses into a list and sorts it based on preference of closest distance or the most optimal route. The user can use the app to navigate to their destinations or send the destinations into Google Maps and follow the navigation route. The application connected to Google Maps API to receive distance of destinations and was sorted using haversine formula to determine the most optimal route for the driver.
